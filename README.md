 * `convert_to_binary.py`. Time to write: 8:24. Usage (Ipython): `convert_to_binary(integer, base=2)` (integer bases other than 2 are allowed).

   ```python
   In [1]: import convert_to_binary as C

   In [2]: C.convert(3)
   Out[2]: (3, '11')

   In [3]: C.convert(5)
   Out[3]: (5, '101')

   In [4]: C.convert(8)
   Out[4]: (8, '1000')

   In [5]: C.convert(5, 3)
   Out[5]: (5, '12')

   In [6]: C.convert(39, 19)
   Out[6]: (39, '21')
   ```
